Description

viterbi is a program written in C++ which, in short, tries to guess words using a Viterbi-style parsing algorithm based on naive unigram frequencies derived from a training corpus.

Installation

Download the source using the link above and uncompress it. A makefile is contained, so all that is required to compile it is to run make in the source directory. The executable is called viterbi.

Usage

For now just run viterbi -h
