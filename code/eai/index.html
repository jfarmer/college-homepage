Description

Extensible Authorship Identiciation (EAI) is a program written in Python and Perl which tries to say, given two authors, to which author a third text is most similar. The default authors are John Goldsmith and Noam Chomsky, mostly for comedic purposes — when I originally wrote this I envisioned this program acting as a backend to something like chomskyornot.com or chomskyvsgoldsmith.com. It requires Numerical Python, NLTK Lite, and probably at least Python 2.3, though I haven't done any real testing with various versions of Python.
