Linux

Linux is a free (in both senses of the word) operating system, and it's the one I use most of the time. At home I run Gentoo Linux and at work I run Ubuntu Linux on my workstation and Red Hat Enterprise Linux on a lot of the servers I help administer. I'm also partial to Macs — OS X is really cool — but I don't use them regularly. Anything related to Linux that I feel the need to share, e.g., shell scripts, themes, and such, you'll find here. Most of it has nothing to do with Linux specifically, but rather UNIX in general.
