LaTeX

LaTeX is an extension of Donald Knuth's TeX typesetting language which allows one to produce logically organized, print-quality documents with relative ease. It's the de facto standard for publication in the scientific and mathematics community. LaTeX is how your professor produces nice-looking handouts for class, or generates a PDF of your calculus homework solutions.

As a student I use LaTeX to typeset all my homework. This has several advantages. First, it just looks nicer. I've seen people try to do math homework in Microsoft Word — I wouldn't subject anyone to something that painful, especially for such poor-looking results. Second, it allows me to archive all my past homework on my computer for easy searching and reference. If I want to quickly find a proof of the Banach-Steinhaus Theorem then I know right where to look. If I get assigned the same homework problem twice in different classes (which happens with surprising regularity) then I just have to copy and paste the solution. In the short-term it requires a bit of work, but it saves me a lot of effort in the long-run. Third, if you continue with mathematics or science, you're probably going to be expected to learn LaTeX at some point. Why not start now?

If you're just beginning with LaTeX then there are plenty of resources on the internet. I recommend the LaTeX primer and the Online tutorials for LaTeX. You can find plenty more by searching Google. You might also want to look at the math section of this page. I include the LaTeX source files for all my homework assignments, and examples are probably the easiest way to get started so long as they're good examples. The idea behind LaTeX is to allow you to type the logical structure of your document and leave the precise formatting details of your document to the LaTeX processor. This is fundamentally different than tools like Microsoft Word, where basically all the decisions you make about your document relate to formatting, e.g., what font sizes to use, where to italicize what, and so on.

I also have my very own homework.sty file which contains several useful definitions and macros that I use in each of my homework assignments (as of March of 2005, anyhow). If you have any questions feel free to contact me.
